Installation
==========

``ofrl`` is available on PyPI, and can be installed from ``pip`` or source as follows.

.. card::
    
    .. tabs::

        .. code-tab:: bash From :class:`pip`

            pip install ofrl

        .. code-tab:: bash From source

            git clone https://github.com/negocia-inc/ofrl
            cd ofrl
            python setup.py install
